# 🏫 智慧学科实验室 · AI高中

> 基于 Dify + DeepSeek 的高中全科 AI 辅助学习微信小程序

## 📖 项目简介

智慧学科实验室是一款面向高中生的 AI 辅助学习微信小程序，致力于打造"虚拟 AI 高中"的创新教育体验。学生可以随时向各学科 AI 老师提问，获得个性化的学习指导，真正做到"随时随地，全科答疑"。

## ✨ 主要功能

- 高考九科 AI 老师：语文、数学、英语、物理、化学、生物、政治、历史、地理
- 小语种：韩语、日语、俄语、德语、法语、西班牙语
- 特色学科：音乐、美术、编导、体育、心理、校医、班主任、留学顾问、科技、模拟人生
- 学情分析：AI 分析知识掌握情况，识别薄弱点
- 对话历史记录与收藏功能

## 🛠 技术架构

- 前端：微信小程序（WXML / WXSS / JavaScript）
- 后端：Dify 平台，每位老师对应一个独立 Agent
- 大模型：DeepSeek（国产大模型）

## 🚀 如何部署

1. 克隆项目
2. 在 Dify 平台为每位老师创建 Agent，接入 DeepSeek
3. 将 app.js 中的 YOUR_API_KEY_HERE 替换为你的 Dify API Key
4. 用微信开发者工具导入项目，填写 AppID 后预览运行

## ⚠️ 注意事项

- API Key 已替换为占位符，需自行配置
- 大模型 API 调用会产生费用，请合理控制使用量
